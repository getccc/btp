# 💱 代币交易功能使用指南

## 功能介绍

代币交易功能允许你在BSC和Solana链上直接进行代币兑换交易，无需离开应用。

### 核心功能

- ✅ BNB ⇄ 任意BEP20代币
- ✅ BEP20代币 ⇄ BEP20代币
- ✅ SOL ⇄ 任意SPL代币
- ✅ SPL代币 ⇄ SPL代币
- ✅ 实时价格报价
- ✅ 滑点设置
- ✅ 自动获取钱包余额
- ✅ 一键最大金额
- ✅ 交易状态追踪

### 增强功能 (新增)

- ✅ **交易历史记录** - 自动保存和查看交易记录
- ✅ **价格图表** - 实时价格走势图 (1H, 4H, 24H, 7D)
- ✅ **路由信息** - 显示最佳交易路径和价格影响
- ✅ **限价单** - 设置目标价格自动执行交易 (Beta)
- ✅ **多链支持** - 在BSC和Solana之间无缝切换

### 支持的链

- **BSC (Binance Smart Chain)**: 通过PancakeSwap Router V2 ✅
- **Solana**: 通过Jupiter Aggregator V6 ✅

## 🔑 前置要求

### 1. 安装MetaMask钱包

MetaMask是浏览器插件钱包，用于连接BSC网络：

1. 访问: https://metamask.io/
2. 下载并安装Chrome/Firefox/Edge扩展
3. 创建或导入钱包
4. 备份助记词（非常重要！）

### 2. 添加BSC网络到MetaMask

如果你的MetaMask没有BSC网络，添加方法：

**方式1: 自动添加（推荐）**
1. 打开应用，点击"💱 Swap"标签
2. 点击"Connect Wallet"连接钱包
3. 应用会自动提示添加BSC网络
4. 点击"批准"即可

**方式2: 手动添加**
1. 打开MetaMask，点击网络下拉菜单
2. 点击"添加网络" → "手动添加网络"
3. 填写以下信息:
   ```
   网络名称: BSC Mainnet
   RPC URL: https://bsc-dataseed.binance.org/
   Chain ID: 56
   货币符号: BNB
   区块浏览器URL: https://bscscan.com/
   ```
4. 点击"保存"

### 3. 准备资金

确保你的钱包中有:
- **BNB**: 用于支付gas费（建议至少0.01 BNB）
- **要交换的代币**: 根据你的交易需求

## 🚀 使用步骤

### 1. 连接钱包

1. 打开应用: http://localhost:3000
2. 点击顶部导航 **"💱 Swap"** 标签
3. 点击 **"Connect Wallet"** 按钮
4. MetaMask会弹出连接请求，点击 **"下一页"** → **"连接"**
5. 如果不是BSC网络，会提示切换网络，点击 **"切换网络"**

### 2. 选择交易对

**选择源代币（From）:**
1. 点击"From"区域的代币选择按钮
2. 从弹出的代币列表中选择你要支付的代币
3. 支持的代币:
   - BNB (原生代币)
   - WBNB, USDT, USDC, BUSD
   - CAKE, ETH, DAI

**选择目标代币（To）:**
1. 点击"To"区域的代币选择按钮
2. 选择你要获得的代币

**快速切换:**
- 点击中间的 **⇅** 按钮可快速交换From和To代币

### 3. 输入交易金额

**输入金额:**
1. 在"From"输入框中输入要交换的数量
2. 应用会自动获取实时报价
3. "To"区域会显示预计获得的代币数量

**使用MAX按钮:**
1. 点击"From"区域右下角的 **"MAX"** 按钮
2. 自动填入可用余额（如果是BNB会预留gas费）

**查看余额:**
- "From"区域右上角显示当前代币余额

### 4. 设置滑点

滑点容忍度决定了你能接受的价格波动范围：

**预设选项:**
- **0.1%**: 适合稳定币交易（USDT⇄USDC）
- **0.5%**: 适合主流代币（推荐）
- **1.0%**: 适合波动性较大的代币
- **3.0%**: 适合流动性较低的代币

**说明:**
- 滑点越低，价格保护越严格，但交易失败风险越高
- 滑点越高，交易成功率越高，但可能以更差的价格成交

### 5. 执行交易

1. 确认所有信息无误后，点击 **"Swap"** 按钮

2. **如果是代币→BNB 或 代币→代币:**
   - 第一次交易会先弹出"授权"请求
   - 点击MetaMask中的"确认"授权代币使用
   - 等待授权交易确认
   - 然后会弹出实际交易请求

3. **MetaMask交易确认:**
   - 检查gas费用（通常0.0003-0.001 BNB）
   - 点击"确认"发送交易
   - 或点击"拒绝"取消交易

4. **等待交易确认:**
   - 交易提交后会显示"Swapping..."状态
   - BSC链通常3-10秒确认
   - 确认后显示成功消息和交易哈希

5. **查看交易详情:**
   - 点击 "View on BscScan →" 链接
   - 在区块链浏览器查看完整交易信息

### 6. 交易完成

- 页面会自动刷新余额
- 新的代币会出现在你的钱包中
- 输入框会清空，准备下一笔交易

## 💡 使用技巧

### 1. 节省Gas费

- **合并交易**: 一次交换更多代币，而不是多次小额交易
- **避开高峰**: BSC网络在凌晨时段gas费通常较低
- **调整gas设置**: 如不急可以在MetaMask中手动降低gas price

### 2. 最佳交易时机

- **查看价格**: 在"Token Rankings"页面查看代币实时价格
- **监控地址**: 在"Address Monitor"页面监控大户动向
- **避免抢跑**: 不要在新代币发布时立即购买（可能是诈骗）

### 3. 安全建议

- **验证代币地址**: 确保选择的是正确的代币（不是假币）
- **小额测试**: 第一次交易建议小额测试
- **检查授权**: 定期检查代币授权，撤销不必要的授权
- **保护私钥**: 永远不要分享助记词和私钥

### 4. 处理交易失败

**常见失败原因:**

1. **"Insufficient balance"** (余额不足)
   - 确保有足够的源代币
   - 预留足够的BNB支付gas费

2. **"Transaction reverted"** (交易回滚)
   - 增加滑点容忍度（改为1%或更高）
   - 价格变化太快，等待片刻重试

3. **"Insufficient liquidity"** (流动性不足)
   - 该交易对流动性太低
   - 尝试减少交易金额
   - 或选择其他交易对

4. **"User rejected transaction"** (用户拒绝)
   - 你在MetaMask中点击了"拒绝"
   - 重新发起交易即可

## ⚠️ 常见问题

### 1. 为什么需要两次确认？

**第一次: 授权**
- 允许PancakeSwap Router使用你的代币
- 只需授权一次，后续交易不需要

**第二次: 交易**
- 实际执行代币交换
- 每次交易都需要确认

### 2. 授权会花费多少gas？

- 授权交易通常花费 **0.0003-0.0005 BNB**
- 实际swap交易花费 **0.0005-0.001 BNB**

### 3. 交易卡住了怎么办？

1. 打开MetaMask查看交易状态
2. 如果pending时间过长（>5分钟）:
   - 可以通过发送0 BNB给自己加速
   - 或者等待交易自动取消
3. 重新发起交易

### 4. 收到的代币数量少于预期？

这是正常的，原因:
- **价格滑点**: 在交易确认期间价格变化
- **PancakeSwap手续费**: 0.25%交易费
- **实际执行价格**: 可能与报价有轻微差异

### 5. 如何取消授权？

出于安全考虑，建议定期检查授权:

1. 访问: https://bscscan.com/tokenapprovalchecker
2. 连接钱包查看所有授权
3. 撤销不需要的授权

### 6. 交易失败会扣gas费吗？

会的！即使交易失败，gas费也会被扣除，因为:
- Gas费是支付给BSC网络的
- 已经消耗了网络资源
- 无论成功与否都需要支付

## 🛡️ 安全注意事项

### ⚠️ 警告标志

以下情况请谨慎:

- 🚫 代币价格暴涨暴跌
- 🚫 新发布的代币（可能是诈骗）
- 🚫 流动性极低的代币
- 🚫 要求授权"无限额度"的代币
- 🚫 社交媒体上的"推荐"代币

### ✅ 安全实践

1. **小额测试**: 第一次交易用小额测试
2. **验证合约**: 在BscScan验证代币合约代码
3. **检查流动性**: 选择流动性池大的代币
4. **保护私钥**: 使用硬件钱包存储大额资金
5. **定期审计**: 检查并撤销不必要的授权
